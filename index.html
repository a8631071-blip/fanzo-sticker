隱藏頁｜工程日誌

v9.3｜2026-05-01｜去星星＋魔術棒版
- 依指定命名為「0501去背工具V9.3」。
- 補回去背區「魔術棒取色」功能：啟用後可在右側格線預覽點擊背景，自動更新目標顏色。
- 魔術棒只做取樣輔助，不改 Web Worker 去背核心、不重寫切割與 ZIP 流程。
- 取色來源使用乾淨取樣畫布，不讀取橘色格線、裁邊遮罩或格子編號，避免取錯色。
- 依取樣色自動建議 tolerance / despill：綠色偏高容許度＋despill；白底、黑底使用較低容許度。

v9.2｜2026-05-01｜去星星整合版
- 以 v9.1 去背工具為主體，新增 Step 2「去星星前處理」模組。
- 整合 Gemini 星星去除工具核心：固定遮罩 bg_48.png / bg_96.png + white alpha unmixing。
- 規則明確化：單張目前預覽圖可直接下載，也可匯入去背流程；批次僅支援 ZIP 下載。
- 新增 3-step 導覽：上傳 → 去星星 → 去背／切割／打包。
- 保留原 v9.1 的去背、切格、Main / Tab、ZIP 打包核心流程，未重寫 Web Worker 演算法。
- 新增去星星區檔案清單、遮罩拖曳、位置微調、匯入去背流程按鈕。
- 本版僅完成靜態整合與程式碼層級檢查，未宣稱完整實機測試。
